# design-studio/ — throwaway mockups (NOT production code)

Sandbox for the Redesign Studio session (2026-07). Everything here is a disposable
HTML mockup used to judge design options by eye before anything is built in
site/src. Nothing imports from or into the app. Safe to delete after the
redesign ships. Decisions of record live in REDESIGN-SPEC.md.

Serve locally: any static server at this folder (e.g. `npx serve design-studio`).
